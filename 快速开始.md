## ⚡ 最快部署方式（推荐）

### 第一步：安装 Git（仅首次需要）

1. 下载 Git：https://git-scm.com/download/win
2. 双击安装，一路点击 "Next"
3. 安装完成后，重启 PowerShell

### 第二步：获取 GitHub Token（仅首次需要）

1. 登录 GitHub：https://github.com
2. 访问：https://github.com/settings/tokens
3. 点击 "Generate new token (classic)"
4. 设置名称：`HeYiYao_Deploy`
5. 勾选权限：`repo`
6. 点击 "Generate token"
7. **复制生成的 Token**（只显示一次！）

### 第三步：运行部署脚本

1. **首次部署**：双击运行 `首次部署.bat`
   - 输入 GitHub 用户名：`MrZh2025`
   - 输入邮箱
   - 推送时粘贴 Token

2. **后续更新**：双击运行 `快速部署.bat`
   - 输入更新说明
   - 自动推送

### 第四步：启用 GitHub Pages（仅首次需要）

1. 访问：https://github.com/MrZh2025/HeYiYao_Public/settings/pages
2. Source 选择：`GitHub Actions`
3. 保存

### 第五步：访问网站

等待 2-5 分钟后，访问：https://MrZh2025.github.io/HeYiYao_Public/

## ❓ 常见问题

### Q1: 推送时要求输入密码？

**A**: 使用 Personal Access Token，不是 GitHub 登录密码。

### Q2: 网站没有更新？

**A**: 
1. 等待 5 分钟
2. 清除浏览器缓存（Ctrl + F5）
3. 检查 Actions 是否部署成功

### Q3: 部署失败？

**A**: 
1. 查看 Actions 页面的错误日志
2. 确认 GitHub Pages 设置正确
3. 检查文件是否有语法错误

### Q4: Token 过期了？

**A**: 
1. 重新生成 Token
2. 下次推送时使用新 Token
